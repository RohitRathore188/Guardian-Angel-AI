# 🛡️ Guardian Angel AI
### AI-Powered Child Rescue & Emergency Response Platform

## 🚀 Overview

Guardian Angel AI is an AI-powered emergency response platform designed to help rescue abandoned, missing, injured, or vulnerable children by connecting citizens, police, hospitals, NGOs, volunteers, and child welfare agencies through one intelligent command platform.

The system uses Artificial Intelligence, GIS mapping, real-time incident management, and multi-role dashboards to coordinate emergency rescue operations.

# 🎯 Problem Statement

Every year thousands of children go missing or remain unattended after accidents and disasters.

Current emergency response systems are fragmented because

- Citizens don't know whom to contact.
- Agencies work independently.
- Rescue coordination is slow.
- Information is scattered.
- No intelligent prioritization exists.

Guardian Angel AI solves this by creating one unified AI-powered emergency command platform.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/RohitRathore188/Guardian-Angel-AI.git
```

Go to project

```bash
cd Guardian-Angel-AI
```

Install dependencies

```bash
npm install
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```
